# E-Commerce Sales & Customer Analytics Dashboard

## Project Overview
This project analyzes e-commerce sales data to generate business insights, customer segmentation, and revenue forecasts.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Prophet
- SQL
- Power BI

## Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- RFM Customer Segmentation
- Sales KPI Dashboard
- Product Performance Analysis
- Country-wise Revenue Analysis
- Revenue Forecasting using Prophet

## Key Insights
- Total Revenue: 8.28M
- Total Orders: 22K
- Total Customers: 4K
- Top Country: United Kingdom
- Top Product: REGENCY CAKESTAND 3 TIER
- Revenue Peak: November 2011

## Dashboard
Interactive Power BI dashboard with:
- KPI Cards
- Revenue Trend
- Customer Segmentation
- Product Analysis
- Country Analysis
- Forecast Analysis

## Forecasting
Used Facebook Prophet to predict future revenue trends and support business planning.
